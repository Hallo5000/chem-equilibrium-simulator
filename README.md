An attempt to simulate chemical equilibrium reactions in an extremely simplified but visually appealing way.

### Dependencies and Build Tools
- Eclipse Temurin 23.0.2
- Gradle Wrapper 8.13
- based on JavaFX 25.0.2 (used SceneBuilder)
